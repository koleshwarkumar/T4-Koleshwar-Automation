# T4-Koleshwar-Automation
Praavah Consulting Assignment for Automation


1.Software is installed in your system:
2.Java (JDK 8 or above)
3.Selenium WebDriver 4
4.IDE (Eclipse / IntelliJ)
5.Chrome Browser
6.Stable Internet Connection

The following steps that we performed:
1. Login Feature
First we open the browser.
Then we enter the application URL.
Then we enter valid email id and password.
Then we click on Login button.
After login we verify dashboard page is displayed.

2. Add New Generator Feature
Then we click on "Generator Management".
Then we click on "Add New Generator" button.
Then we enter Generator Name.
Then we enter Internal Account Number.
Then we click on Save button.
Then we verify generator is created successfully.

3. Fill Billing Information Feature
Then we select Industry from dropdown.
Then we enter Street Address.
Then we enter Email address.
Then we enter Phone number.
Then we save the billing information.

4. Generate and Verify Latitude/Longitude Feature
Then we click on Generate Latitude/Longitude button.
Then system auto-fills latitude and longitude values.
Then we verify that both fields are not empty.

5.Copy Billing Information Feature
Then we click on "Copy Billing Information" button.
Then we verify copied data is correctly displayed.

6.Fill Service Hours Feature
Then we enter Open time.
Then we enter Lunch time.
Then we enter Close time.
Then we click on "Copy Hours to All Days".
Then we verify hours applied to all days.

7. Route Assignment Feature
Then we click on Route Assignment.
Then we click on Add Service.
Then we select Route name (Example: My name Route).
Then we select Service Type as "Medical Waste".
Then we select Frequency as "Multiple Times Weekly".
Then we select weekdays (Thu / Fri / Sat / Sun).
Then we select Scope of Work (2 containers).

Then we save the service details.

8. Price Book Update Feature
Then we open Price Book section.
Then we enter service price.
Then we update the prices.
Then we click on Save button.

9. Logout Feature
Then we click on Profile icon.
Then we click on Logout button.
Then we verify login page is displayed again.
